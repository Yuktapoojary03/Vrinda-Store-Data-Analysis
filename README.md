# 📊 Vrinda Store Data Analysis Dashboard

An interactive Excel dashboard developed to analyze Vrinda Store sales performance, customer behavior, and business trends. The project transforms raw sales data into meaningful insights using Pivot Tables, Pivot Charts, Slicers, and KPI metrics, enabling data-driven decision-making.

## 🔹 Features

* Sales performance analysis
* Order and revenue tracking
* Customer demographics analysis
* Monthly and yearly trend visualization
* Product category performance insights
* Interactive filtering using slicers
* KPI-based reporting dashboard

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting
* Excel Functions & Formulas

## 📈 Key Insights

* Identified top-performing products and categories
* Analyzed customer purchasing patterns
* Tracked sales trends across different periods
* Evaluated revenue and order distribution
* Generated actionable business insights through visual reports

## 🎯 Objective

To create an interactive and user-friendly Excel dashboard that helps stakeholders monitor sales performance, understand customer behavior, and support strategic business decisions through data visualization and analysis.
